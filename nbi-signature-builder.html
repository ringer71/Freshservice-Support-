📧 How to add to Outlook
  1. Fill in your details above
  2. Click Copy Signature for Outlook
  3. Open Outlook → File → Options → Mail → Signatures…
  4. Click New, name it (e.g. NBI Standard)
  5. Click inside the edit box → press Ctrl+V
  6. Set as default for New messages & Replies
  7. Click OK
⚠ If Ctrl+V pastes as plain text, right-click → Paste Special → HTML Format.

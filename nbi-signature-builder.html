How to Add Your Signature in NetSuite

  1. Click Download Signature File. A file named nbi-netsuite-signature.txt will save to your Downloads folder.
  2. Open the downloaded file in Notepad (Windows) or TextEdit (Mac). Press Ctrl+A (or Cmd+A on Mac) to select all, then Ctrl+C (Cmd+C) to copy.
  3. In NetSuite, go to Home → Set Preferences → General.
  4. Scroll to the User Profile section. Click into the Signature box.
  5. Delete anything currently in the box, then paste (Ctrl+V / Cmd+V).
  6. Tick Add Signature to Messages.
  7. Click Save at the top of the page.
  8. Send a test email from NetSuite (e.g. preview a transaction email) to confirm the signature renders correctly.

If the signature appears as plain text or shows tags like <br />, the paste didn't work correctly. Clear the field, copy from the file again, and re-paste.

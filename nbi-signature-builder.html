How to Upload to NetSuite

  1. Click Download HTML Signature. A file named nbi-signature.html will save to your Downloads folder.
  2. In NetSuite, go to Home → Set Preferences → General.
  3. Scroll to the Signature section and tick Use Signature.
  4. Open the downloaded HTML file in Notepad (Windows) or TextEdit (Mac), select all, copy, and paste into the NetSuite signature box.
  5. Save preferences. Send a test email to yourself to confirm.

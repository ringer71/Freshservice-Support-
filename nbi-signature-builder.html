📧 How to install in Outlook
1
Fill in your details and click Download Signature File
2
Open File Explorer and paste this path into the address bar:
%APPDATA%\Microsoft\Signatures
Press Enter to navigate there
3
Move the downloaded .htm file into that folder
4
Open Outlook → File → Options → Mail → Signatures…
5
Your signature will appear in the list — set it as default for New messages and Replies → click OK
⚠ Outlook must be restarted after saving the file if it was already open.

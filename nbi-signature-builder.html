📧 How to install in Outlook
1
Fill in your details and click Download Signature File
2
Open File Explorer, paste this into the address bar and press Enter:
%APPDATA%\Microsoft\Signatures
3
Move the downloaded NBI Standard.htm file into that folder
4
Open Outlook → File → Options → Mail → Signatures…
5
Select NBI Standard, set as default for New messages & Replies → OK
⚠ Restart Outlook after saving the file if it was already open.
